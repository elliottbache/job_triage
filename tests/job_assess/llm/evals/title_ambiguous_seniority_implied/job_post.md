We are looking for a software engineer to build and own backend services used by internal and customer-facing applications. Strong PostgreSQL schema design and query optimization skills are required. Experience building and maintaining REST APIs in production is important. Docker and CI/CD experience are preferred. This is a fully remote role working across distributed teams.
